# My Tasks App ✨

A beautiful to-do list app built with React.

🔗 **Live Demo:** [Click here to open](https://marbi26.github.io/to_do_list)

## Features
- Add, complete & delete tasks
- Due dates, priority levels & tags
- Smooth animations
- Nude color theme with nature banner

## Run Locally
```bash
npm install
npm start
```

## Built With
- React
- GitHub Pages
